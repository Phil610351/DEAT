# DEAT
Achieving Fast Environment Adaptation of DRL-Based Computation Offloading in Mobile Edge Computing
## Usage:
To train DDPG:
```bash
python main.py --algorithm dc_drl
```
To train MADDPG:
```bash
python main.py --algorithm maddpg
```
To train DC-DRL:
```bash
python main.py --algorithm dc_drl
```
To train DEAT:
```bash
python main.py --algorithm deat
```
